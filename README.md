# TDD Objects

This problem set is all about objects!

## Instructions

Complete each function in `index.js` according to the JSDoc.

Run `npm test` in your terminal to test your code. Try to pass all test cases!

> [!TIP]
>
> You'll need to use everything you've learned so far to write these functions!
> Think about how to work together with conditionals, loops, arrays, and objects.
